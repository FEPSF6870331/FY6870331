Abstract

This paper presents a novel adaptive learning strategy. The most accurate classification classroom learning model from a hetrogenous classifer pool is selected at each learning epoch. Our goal is to create greater incremental improvment in accuracy, compared to using a single classification model throughout the process. The strategy is evaluated on beanchmark data sets. We further consider an application to texture classification in sonar mosaic imagry and compare it with non-adaptive active learning and traditional machine learning approches.
